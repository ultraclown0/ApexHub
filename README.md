# ApexHub

A non-commercial, open-source web app for watching **Apex Legends tournament
broadcasts** — several Twitch streams at once (official broadcast + player POVs)
— alongside **match statistics** for teams and players. Desktop-first,
responsive, bilingual (English / Russian).

> Hobby project. Not affiliated with EA or Respawn Entertainment.
